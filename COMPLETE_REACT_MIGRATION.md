## 🚀 Démarrage

### Installation
```bash
cd lectorium-frontend
npm install
```

### Configuration
Le fichier `.env` est déjà créé avec :
```env
VITE_API_URL=http://localhost:8000/api
```

### Lancer
```bash
npm run dev
```

Ouvrir : **http://localhost:3000**
